### 1. Persiapan Laragon
*   Buka **Laragon**.
*   Klik **Start All** (Wajib aktifkan MySQL).

### 2. Cara Menjalankan (Paling Gampang)
Cukup jalankan satu perintah di terminal utama VS Code:
1.  Buka terminal (`Ctrl` + `~`).
2.  Jalankan: `npm install` (Hanya perlu sekali).
3.  Jalankan: `cd backend && npm install && cd ..` (Hanya perlu sekali).
4.  Jalankan: **`npm run dev`**
5.  Selesai! Terminal akan otomatis menyalakan **Frontend (Vite)** dan **Backend (Express)** sekaligus.
